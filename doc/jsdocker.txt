INTRODUCTION						*jsdocker-introduction*

JsDocker helps you to generate comments for JsDoc.
You can use this like the followings:
>
	:JsDockerAddJsDoc
<

For example, if the cursor is placed on the first line of the following code
and you execute `:JsDockerAddJsDoc`,
>
	function foo(bar, baz, qux) {
		// some code
	}
<

then you get the result as below.
>
	/**
	 * @param {} bar
	 * @param {} baz
	 * @param {} qux
	 * @return {}
	 */
	function foo(bar, baz, qux) {
		// some code
	}
<

COMMANDS						*jsdocker-commands*

							*:JsDockerAddJsDoc*
:JsDockerAddJsDoc
	Generate JsDoc comment for the function described in current line.

KEY MAPPINGS						*jsdocker-key-mappings*

Normal mode mappings.

<Plug>(jsdocker-add-jsdoc)			*<Plug>(jsdocker-add-jsdoc)*
	Generate JsDoc comment for the function described in current line.
